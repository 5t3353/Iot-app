Air quality I.O.T system

database dashboard

This Proyect

This web corresponds to the result of a university project of the data networks course given at the andres bello university, which consists of making an I.O.T. application using the knowledge acquired about networks and internet. the application consists of a system in charge of monitoring the air quality of an enclosure, whether domestic, industrial, etc. The variables to be monitored are temperature, humidity and carbon dioxide present in the air of the place, as well as a machine learning model is integrated to make predictions and feedback the system with respect to the patterns learned with the data once stored, this feedback affects to operate a device remotely, so that it can, in addition to monitoring, control the air quality in the place with respect to what is inferred from the data obtained from the sensors.
